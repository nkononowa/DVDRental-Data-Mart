# DVDRental-Data-Mart
Проектирование и реализация **витрины данных** на основе учебной БД **dvdrental** в PostgreSQL.

## Используемые инструменты

| Компонент | Описание |
|------------|-----------|
| **PostgreSQL 16** | Основная СУБД |
| **DBeaver** | Клиент SQL и инструмент разработки |
| **Docker** | Контейнеризация PostgreSQL |
| **Redash** | BI |

---

## Архитектура проекта

Проект организован в виде трёхуровневой схемы:
stg → raw → dm → cache(matviews)
| Схема | Назначение |
|--------|-------------|
| `stg` | Сырые данные, импортированные из исходного дампа |
| `raw` | Очистка и нормализация данных |
| `dm` | Размерно-фактовая модель витрины |
| `etl` | Таблицы и скрипты управления загрузками (контроль, валидация) |

На основе данных витрины был создан **дашборд** для анализа основных показателей проката фильмов и динамики доходов.
